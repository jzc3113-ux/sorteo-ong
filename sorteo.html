Tú dijiste:
Quiero hacer una pagina en virtual para poder hacer sorteos de mi ong para abartar costo de compra solo quiero que pueda enviar del excel importarlo a la pagina y que salga el ganador
ChatGPT Plus

¡Perfecto! Para abaratar costos y mantenerlo simple, hagámoslo 100% en el navegador (sin backend, hosting estático tipo GitHub Pages/Netlify). Subes un Excel con la lista de participantes y la página elige al ganador con un RNG criptográfico.

Aquí tienes un archivo único (sorteo.html) que funciona tal cual.

  • Acepta .xlsx, .xls y .csv.

  • Toma la columna que elijas (por ejemplo “Nombre” o “Email”).

  • Quita vacíos y duplicados (opcional).

  • Muestra el ganador y te permite exportar un acta del sorteo (CSV).

Guárdalo como sorteo.html y ábrelo en el navegador.

html
<!doctype html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <title>Sorteo ONG</title>
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <!-- Librería para leer Excel en el navegador -->
  <script src="https://cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js"></script>
  <style>
    :root { font-family: system-ui, Arial, sans-serif; }
    body { margin: 24px; }
    h1 { margin: 0 0 8px; }
    .card { border: 1px solid #ddd; border-radius: 12px; padding: 16px; margin: 16px 0; }
    label { font-weight: 600; display:block; margin: 8px 0 4px; }
    input[type="file"] { padding: 8px; }
    select, input[type="checkbox"], button { margin-top: 6px; }
    button { padding: 10px 14px; border-radius: 8px; border: 1px solid #ccc; cursor: pointer; }
    button.primary { border-color: #333; }
    .grid { display: grid; grid-template-columns: 1fr; gap: 12px; }
    .winner { font-size: 1.6rem; font-weight: 800; margin-top: 12px; }
    .muted { color: #666; font-size: .9rem; }
    .pill { display:inline-block; padding: 2px 8px; border:1px solid #ddd; border-radius: 999px; margin-right:6px; }
    .list { max-height: 220px; overflow:auto; border:1px solid #eee; border-radius:8px; padding:8px; }
    .ok { color: #0a7; }
    .warn { color: #b60; }
    .err { color: #b00; }
  </style>
</head>
<body>
  <h1>🎉 Sorteo ONG</h1>
  <p class="muted">Carga tu Excel/CSV con la lista de participantes, elige la columna y realiza el sorteo.</p>

  <div class="card">
    <div class="grid">
      <div>
        <label>Archivo (.xlsx, .xls, .csv)</label>
        <input id="file" type="file" accept=".xlsx,.xls,.csv" />
        <div id="fileInfo" class="muted"></div>
      </div>

      <div>
        <label>Hoja (solo Excel)</label>
        <select id="sheetSelect" disabled></select>
      </div>

      <div>
        <label>Columna a usar</label>
        <select id="columnSelect" disabled></select>
      </div>

      <div>
        <label><input type="checkbox" id="dedupe" checked /> Eliminar duplicados</label>
        <label><input type="checkbox" id="trimSpaces" checked /> Limpiar espacios en blanco</label>
        <label><input type="checkbox" id="ignoreHeader" /> Ignorar primera fila como encabezado</label>
      </div>

      <div>
        <button id="previewBtn">Previsualizar</button>
        <button id="drawBtn" class="primary" disabled>🎲 Elegir ganador</button>
        <button id="exportBtn" disabled>⬇️ Exportar acta</button>
      </div>
    </div>
  </div>

  <div class="card">
    <div><span class="pill" id="count">0 participantes</span>
         <span class="pill" id="seedInfo">Semilla: crypto</span></div>
    <div id="status" class="muted">Sin datos cargados.</div>
    <div id="winner" class="winner"></div>
    <div id="list" class="list"></div>
  </div>

<script>
  const $ = (id) => document.getElementById(id);
  const state = {
    workbook: null,
    rows: [],
    values: [],
    winner: null,
    fileName: null,
    pickedAt: null,
    columnName: null,
    sheetName: null
  };

  $('file').addEventListener('change', handleFile);
  $('previewBtn').addEventListener('click', preview);
  $('drawBtn').addEventListener('click', draw);
  $('exportBtn').addEventListener('click', exportCSV);
  $('sheetSelect').addEventListener('change', onSheetChange);

  function setStatus(msg, cls='muted') {
    const s = $('status'); s.textContent = msg; s.className = cls;
  }

  function handleFile(evt) {
    const file = evt.target.files[0];
    state.fileName = file ? file.name : null;
    $('winner').textContent = '';
    $('list').innerHTML = '';
    $('columnSelect').innerHTML = '';
    $('sheetSelect').innerHTML = '';
    $('columnSelect').disabled = true;
    $('sheetSelect').disabled = true;
    $('drawBtn').disabled = true;
    $('exportBtn').disabled = true;

    if (!file) return;

    $('fileInfo').textContent = `Archivo: ${file.name} (${Math.round(file.size/1024)} KB)`;

    const reader = new FileReader();
    const isCSV = file.name.toLowerCase().endsWith('.csv');

    reader.onload = (e) => {
      try {
        if (isCSV) {
          // Parse CSV simple
          const text = e.target.result;
          const rows = text.split(/\r?\n/).map(r => r.split(','));
          state.workbook = null;
          state.rows = rows;
          // Construir columnas por índice
          buildColumnSelectFromRows(rows);
          setStatus('CSV cargado. Elige columna y previsualiza.', 'ok');
          $('columnSelect').disabled = false;
          $('sheetSelect').disabled = true;
        } else {
          const data = new Uint8Array(e.target.result);
          const wb = XLSX.read(data, {type:'array'});
          state.workbook = wb;
          // Poblar hojas
          $('sheetSelect').innerHTML = '';
          wb.SheetNames.forEach((name, i) => {
            const opt = document.createElement('option');
            opt.value = name; opt.textContent = name + (i===0?' (sugerida)':'');
            $('sheetSelect').appendChild(opt);
          });
          $('sheetSelect').disabled = false;
          onSheetChange();
          setStatus('Excel cargado. Selecciona hoja y columna, luego previsualiza.', 'ok');
        }
      } catch (err) {
        console.error(err);
        setStatus('Error al leer el archivo. Verifica el formato.', 'err');
      }
    };

    if (isCSV) reader.readAsText(file, 'utf-8');
    else reader.readAsArrayBuffer(file);
  }

  function onSheetChange() {
    if (!state.workbook) return;
    const sheetName = $('sheetSelect').value || state.workbook.SheetNames[0];
    state.sheetName = sheetName;
    const ws = state.workbook.Sheets[sheetName];
    const rows = XLSX.utils.sheet_to_json(ws, {header:1, defval:''});
    state.rows = rows;
    buildColumnSelectFromRows(rows);
    $('columnSelect').disabled = false;
  }

  function buildColumnSelectFromRows(rows) {
    $('columnSelect').innerHTML = '';
    const maxCols = Math.max(...rows.map(r => r.length), 0);
    const header = rows[0] || [];
    for (let i = 0; i < maxCols; i++) {
      const nameGuess = header[i] || `Columna ${i+1}`;
      const opt = document.createElement('option');
      opt.value = i; opt.textContent = nameGuess;
      $('columnSelect').appendChild(opt);
    }
  }

  function sanitize(values) {
    const trim = $('trimSpaces').checked;
    const ignoreHeader = $('ignoreHeader').checked;

    let v = [...values];
    if (ignoreHeader) v = v.slice(1);
    v = v.map(x => (x==null? '': String(x))).map(x => trim ? x.trim() : x);
    v = v.filter(x => x.length > 0);

    if ($('dedupe').checked) {
      const seen = new Set();
      v = v.filter(x => (seen.has(x) ? false : (seen.add(x), true)));
    }
    return v;
  }

  function preview() {
    if (!state.rows.length) { setStatus('Primero carga un archivo.', 'warn'); return; }
    const colIdx = parseInt($('columnSelect').value, 10) || 0;
    state.columnName = $('columnSelect').selectedOptions[0]?.textContent || `Columna ${colIdx+1}`;

    const col = state.rows.map(r => r[colIdx]);
    const values = sanitize(col);
    state.values = values;

    $('count').textContent = `${values.length} participantes`;
    $('list').innerHTML = values.map(v => `<div>${escapeHTML(v)}</div>`).join('');
    $('winner').textContent = '';
    state.winner = null;
    state.pickedAt = null;

    if (values.length === 0) {
      setStatus('No hay participantes válidos en esa columna.', 'warn');
      $('drawBtn').disabled = true;
      $('exportBtn').disabled = true;
    } else {
      setStatus('Lista lista. ¡Puedes realizar el sorteo!', 'ok');
      $('drawBtn').disabled = false;
      $('exportBtn').disabled = true;
    }
  }

  function draw() {
    if (!state.values.length) { setStatus('No hay participantes para sortear.', 'warn'); return; }
    // RNG criptográfico para imparcialidad
    const idx = cryptoRandomInt(0, state.values.length - 1);
    const winner = state.values[idx];
    state.winner = winner;
    state.pickedAt = new Date().toISOString();

    $('winner').textContent = `GANADOR: ${winner}`;
    setStatus(`Ganador seleccionado de ${state.values.length} participantes.`, 'ok');
    $('exportBtn').disabled = false;

    // Resaltar ganador en la lista
    const children = $('list').children;
    for (let i = 0; i < children.length; i++) {
      children[i].style.fontWeight = (i === idx) ? '800' : '400';
    }
  }

  // Genera entero aleatorio usando Web Crypto: [min, max]
  function cryptoRandomInt(min, max) {
    const range = max - min + 1;
    if (range <= 0) return min;
    // Eliminar sesgo: rejection sampling
    const maxUint = 0xFFFFFFFF;
    const limit = Math.floor(maxUint / range) * range;
    let rand;
    do {
      const buf = new Uint32Array(1);
      crypto.getRandomValues(buf);
      rand = buf[0];
    } while (rand >= limit);
    return min + (rand % range);
  }

  function exportCSV() {
    if (!state.winner) { setStatus('Primero realiza el sorteo.', 'warn'); return; }
    const lines = [];
    lines.push('fecha_iso,archivo,hoja,columna,participantes,ganador');
    const row = [
      state.pickedAt,
      state.fileName || '',
      state.sheetName || '',
      state.columnName || '',
      state.values.length,
      `"${state.winner.replace(/"/g,'""')}"`
    ].join(',');
    lines.push(row);

    // Adjuntar lista completa (opcional, útil para auditoría)
    lines.push('');
    lines.push('lista_completa');
    state.values.forEach(v => lines.push(`"${String(v).replace(/"/g,'""')}"`));

    const blob = new Blob([lines.join('\n')], {type: 'text/csv;charset=utf-8;'});
    const a = document.createElement('a');
    a.href = URL.createObjectURL(blob);
    const ts = new Date().toISOString().replace(/[:.]/g,'-');
    a.download = `acta-sorteo-${ts}.csv`;
    document.body.appendChild(a);
    a.click();
    URL.revokeObjectURL(a.href);
    a.remove();
  }

  function escapeHTML(s) {
    return s.replace(/[&<>"']/g, c => ({'&':'&amp;','<':'&lt;','>':'&gt;','"':'&quot;',"'":'&#039;'}[c]));
  }
</script>
</body>
</html>

Cómo preparar tu Excel

  • Una sola columna con los participantes (ej. “Nombre” o “Email”).

  • Fila 1 puede ser encabezado (activa “Ignorar primera fila”).

  • Evita filas completamente vacías.

Cómo publicarlo gratis

  1. Sube sorteo.html a un repositorio en GitHub.

  2. Activa GitHub Pages (Settings → Pages → Source: “Deploy from a branch”).

  3. Abre la URL que te da GitHub Pages y úsalo desde cualquier navegador.

Extras que puedo añadir si te sirve

  • Múltiples ganadores y suplentes.

  • Pesos por número de tickets.

  • Verificación pública con “semilla” (provably fair).

  • Branding de tu ONG y acta en PDF.

Si quieres, dime si necesitas múltiples ganadores o tickets por persona y te lo dejo listo en el mismo archivo.

Te explico paso a paso dónde y cómo subirlo a GitHub para que quede online gratis con GitHub Pages.

1. Crear una cuenta en GitHub

Si no tienes:

  • Ve a https://github.com

  • Regístrate con tu email.

2. Crear un repositorio

  1. En la esquina superior derecha, haz clic en el botón + → New repository.

  2. Ponle un nombre, por ejemplo:

    sorteo-ong

  3. Marca Public (necesario para que GitHub Pages lo muestre).

  4. No es necesario agregar README ni nada más por ahora (pero si quieres, puedes marcarlo).

  5. Haz clic en Create repository.

3. Subir el archivo sorteo.html

Tienes dos formas:

Opción rápida (desde el navegador)

  1. Dentro del repositorio, haz clic en Add file → Upload files.

  2. Arrastra tu archivo sorteo.html.

  3. Haz clic en Commit changes (guardar cambios).

4. Activar GitHub Pages

  1. En tu repositorio, ve a Settings (pestaña arriba).

  2. En el menú lateral, busca Pages (o escribe “Pages” en el buscador de settings).

  3. En la sección Build and deployment:

    • Source: selecciona Deploy from a branch.

    • Branch: elige main (o master) y la carpeta / (root).

  4. Guarda los cambios.

  5. GitHub generará tu página en unos segundos.
    Te mostrará una URL tipo:

    arduino
    https://tuusuario.github.io/sorteo-ong

5. Probarlo

  • Abre esa URL y ya podrás usar tu página de sorteos.

  • Cada vez que subas un cambio, GitHub Pages se actualizará automáticamente.

Si quieres, puedo hacerte una mini guía con capturas de pantalla para que no te pierdas en ningún paso.
